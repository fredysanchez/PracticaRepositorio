Cuales son los pasos para mostrar una pagina web

  1. Se genera un flujo de informacion donde el protocolo IP tiene que empaquetarla y poner una etiquete(IP origen, Ip destino ,Tipo de paquete) y ademas una etiqueta para el proxy.
  2. El paquete es enviado a la red corporativa donde se encuentra en router que en caso de que sea necesario cambia de ruta a otra red
  3. El paquete llega al proxy y es analizado mediante la abstraccion de la URL y si es permitible la deja caso contrario destruye el paquete.
  4. El paquete se dirige al Firewall corporativo
  5. El paquete llega a internet donde busca la informacion
  6. La informacion puede ser accedida unicamente en los puertos 80 y 25
  7. Una vez encontrada la informacion es empaquetado y enviado de vuelta
